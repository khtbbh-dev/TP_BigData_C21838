# TP_BigData_C21838
Contient le raport tp sur les commandes utilisées en cours dans la réalisationde ce TP ainsi que des fichiers scripts permettant de faire le TP automatiquement. 
Le Big_DataManager est un gestionnaire tout-en-un pour tes TP Hadoop. Il regroupe :
-HDFS
-MapReduce
-YARN
-Spark
-Hive
-Gestion des DataNodes
-Analyse des blocs
-Upload CSV / Parquet
-Génération de rapports
